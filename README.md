# Maria Goundry Portfolio

Portfolio website for fashion designer Maria Goundry, showcasing design projects, photoshoots, press features, and available looks for press loans.

## Sections

- **Home**: Portfolio overview and featured work
- **Projects**: Fashion design projects from BA coursework and professional work (2022-2024)
- **Photoshoots**: Professional photoshoot galleries featuring collaborations and campaigns
- **Press**: Press mentions and features in fashion publications
- **Press Loans**: Available looks to borrow for editorial and styling projects

## Technical Details

- Static HTML/CSS/JavaScript website
- Hosted on GitHub Pages
- Responsive design (mobile-first approach)
- 888 portfolio images organized by section
- Minimalist monochromatic design (white background, black text)
- CSS Grid-based layouts

## Local Development

To run locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
